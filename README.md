# MBox
使用方式：</p>
打开Android Studio，Open an existsing Android Studio project，选中下载后解压的目录即可 </p>
</p></p>
一个学习的例子，里面的一些功能是网上拼拼凑凑的非原创，目前已经实现：</p>
1、每个ViewPager子项为一个Fragment，每个子项间支撑点击、滑动效果</p>
2、其中一个Fragment中实现了listview</p>
</p></p>
